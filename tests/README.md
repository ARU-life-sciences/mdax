# tests/

Rust integration tests for the full `mdax` two-pass pipeline. Run with `cargo test`.

A companion Python script `compare_truth.py` runs `mdax` on the 57-read HiFi dataset and writes a per-read comparison TSV for downstream analysis and plotting:

```bash
# from repo root
python3 tests/compare_truth.py                        # writes tests/results_comparison.tsv
python3 tests/compare_truth.py --out results.tsv      # custom output path
```

Output columns: `read_id`, `label`, `truth_decision`, `truth_split`, `truth_arm_len`, `truth_identity`, `truth_gap_est`, `mdax_decision`, `mdax_split`, `split_delta_bp`, `abs_delta_bp`, `pass_fail`.

## Test cases

### `artefact_is_cut_palindrome_is_preserved`

Uses `data/test_foldback.fasta` (5 synthetic reads, generated by `data/synthesise.py`, seed 42):

| Read | Length | Structure | Expected decision |
|------|--------|-----------|-------------------|
| `artefact_foldback` | 6000 bp | arm A + RC(A), unique junction (n=1) | artefact → cut to ~3000 bp |
| `true_palindrome_1` | 6000 bp | arm B + RC(B), shared junction (n=2) | real → kept intact |
| `true_palindrome_2` | 6000 bp | arm B + RC(B), shared junction (n=2) | real → kept intact |
| `normal_read` | 6000 bp | random DNA, no foldback | not detected → passes through |
| `8000bp` | 4946 bp | arm D + RC(D), unique junction (n=1) | artefact → cut to ~2473 bp |

Config: `min_support=2`, `k=17`, `w=21`, `refine_arm=1200`.

### `artefact_read_is_cut_to_single_arm`

Minimal synthetic test: a single 6000 bp read (3000 bp arm + RC) is cut to a single arm (~3000 bp, within ±200 bp tolerance).

### `compound_artefact_is_cut_at_least_once`

A nested foldback (12000 bp = inner arm + RC of inner arm) is cut at least once, producing output ≤ half the input length.

### `real_artefacts_detected_and_cut`

Primary regression test against 57 real PacBio HiFi CCS reads from `ERR12263839` (MDA library, *C. elegans*). Ground truth was independently derived using BLAST self-alignment (see `data/extract_and_characterise.py`).

Read categories (from `data/chris_artefacts_truth.tsv`):

| Label prefix | n | Identity range | Gap | Expected decision |
|---|---|---|---|---|
| `art_hi_*` | 5 | ≥0.995 | 0 | artefact |
| `art_md_*` | 5 | 0.985–0.995 | 0 | artefact |
| `art_lo_*` | 8 | 0.950–0.985 | 0 | artefact |
| `art_vlo_*` | 2 | 0.940–0.950 | 0 | artefact |
| `li_*` | 5 | ~0.78 | 1.6–4 kb | artefact |
| `real_hi_*` | 14 | ≥0.95 | 0 | real |
| `real_lo_*` | 18 | varied | 0 | real |

For each detected artefact read, the test checks that:
- The refined split position is within ±100 bp of the BLAST-determined split.
- The output sequence is cut to approximately the left arm length.
- Identity estimates are in the expected range.

Config: `k=17`, `w=21`, `min_matches=20`, `min_identity=0.6`, `cut_low_ident=true`.

---

## Interpretation

These tests define the correctness contract for the HiFi pipeline.

**Artefact recall.** `real_artefacts_detected_and_cut` shows that `mdax` recovers all five artefact classes — from clean high-identity hairpins (`art_hi_*`, ≥0.995) down to large-gap template switches (`li_*`, ~0.78 identity, 1.6–4 kb gap). The `li_*` reads are the most structurally complex: the junction refinement must locate the arm boundary across a multi-kilobase gap rather than a clean hairpin. Passing them validates the δ-scan and `use_hint` logic.

**False-positive rate.** All 32 `real_*` reads (14 high-identity, 18 lower-identity) pass through unmodified. This is the harder constraint — a false positive deletes genuine genomic sequence.

**Breakpoint accuracy.** Split positions are within ±100 bp of the BLAST-derived ground truth for all detected reads. For a 6–10 kb HiFi read this corresponds to <2% positional error, which is within the assembly-correction tolerance.

**Practical guidance for HiFi data.** The default settings (`--mode balanced --refine-mode hifi --min-identity 0.6 --cut-low-ident`) are validated here. The `min_identity=0.6` threshold is set to capture the `li_*` class (~0.78) with margin while excluding noise.
